# KogamaWorldEdit

Adds WorldEdit-style building commands to KoGaMa

## Installation
1. Download latest nightly build of MelonLoader
2. Extract zip to ```%localappdata%\KogamaLauncher-WWW\Launcher\Standalone\Mods```

## How to use
1. Right click + Ctrl to select pos1 / pos2
2. Type one of the available commands
    - `//set <id/name>` – fills the selected area (e.g. `//set black_concrete`)
   - `//replace <old_material> <new_material>` – replaces materials in the selected area
   - `//stack <count>` – repeats the selected building in view direction
   
> ⚠️ **Warning:** There is no undo function.
